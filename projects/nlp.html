Disaster Tweet Classification

A Natural Language Processing (NLP) project aimed at classifying tweets into two categories: tweets related to real disasters and those that are not. The project utilizes NLP techniques and machine learning to automatically identify tweets containing information about actual disasters, enabling timely response and support efforts.

Key Features:

  1. Text Preprocessing: NLP techniques such as text preprocessing, tokenization, and feature extraction are applied to convert raw text data into a format suitable for machine learning.
  2. Machine Learning Models: Various machine learning algorithms, such as linear SVM, are employed to classify tweets based on their content. These models are trained on labeled data to predict whether a tweet pertains to a real disaster.
  3. Performance Evaluation: The accuracy and performance of the machine learning models are assessed using metrics like accuracy, precision, recall, and F1-score to ensure reliable tweet classification. The image above is a comparison between the accuracy of the different models used.

    • Fun Fact:

    The term disaster tweet classification does not really clarify what this project is about, so I will do it instead. Twitter (now X) is the most popular social network for real-time interaction on topics ranging from cats to national emergencies. Just like the wide variety of discussions, Twitter boasts a varied range of user ages as well. Some popular expressions used on the internet these days are, "That's fire!", "Ooh that burns", "I am shaking!" and so on. The aim of this project is to recognise that when someone tweets "That's lit!", are they talking about something that is literally on fire, constituting an emergency or are they simply amazed by something.
