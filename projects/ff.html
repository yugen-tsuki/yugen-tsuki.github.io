Flatmate Finder

Flatmate Finder is a mobile application designed to simplify the process of finding roommates and suitable apartments. The application streamlines the roommate and apartment search by aggregating data from various apartment listing websites, providing users with a unified interface to browse listings and connect with potential roommates.

Key Features:

  1. Data Aggregation: Flatmate Finder collects and updates apartment listings from multiple sources, ensuring users have access to the latest available options.
  2. User Profiles: Users can create profiles with information about their preferences, budget, and lifestyle. This information is used to match users with compatible roommates and apartments.
  3. Messaging: The application includes a messaging feature that allows users to communicate with potential roommates or apartment owners directly within the app.
  4. Search and Filters: Users can search for apartments based on location, price range, and other criteria. Advanced filters help users narrow down their options.

Tech Stack:

  • Node.js
  • PostgreSQL
  • Sequelize
  • Python
  • Swift

    • Fun Fact:

    This project was built as a solution to a very real problem that all teammates had faced when we moved to the US for higher education. In fact, we conducted a survey and out of 165 of our classmates, 120 had faced the same issues as us: difficulty in finding a good apartment that is both affordable and close to the campus and finding a compatible roommate.
