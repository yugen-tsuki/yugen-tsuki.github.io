Networkit

The project aimed to simulate and understand essential networking concepts by configuring a Cisco Router and a Linux PC as a LAN switch within a virtual environment. The simulation utilized GNS3 for network emulation and Wireshark for packet analysis. Key objectives included configuring the network elements, observing MAC table updates, and implementing a spanning tree protocol for loop-free routing.

Key Features:

  1. Network Configuration: The project involved configuring a Cisco Router and a Linux PC within GNS3 to simulate a LAN switch environment. This allowed for hands-on experience in setting up network components.
  2. MAC Table Updates: During the simulation, the project explored how LAN switches update their MAC tables in response to network activity, showcasing the dynamic nature of Ethernet-based networks.
  3. Spanning Tree Protocol: An integral part of the project was the implementation and observation of a spanning tree protocol. This protocol ensures loop-free routing within the network, enhancing network reliability.
  4. Packet Analysis: Wireshark was employed to capture and analyze network packets, providing insights into network behavior, frame forwarding, and protocol operations.

Tech Stack:

  • Ubuntu
  • GNS3
  • Wireshark
